# План: Додавання сортування карток

## Зібрана інформація

### Структура проекту:

- **CardsContent.tsx** - основний компонент для відображення карток з фільтрами
- **FilterPanel.tsx** - панель фільтрів (сайдбар)
- **api.ts** - API сервіс для взаємодії з бекендом
- **types/index.ts** - TypeScript типи

### Поля картки (WineCard):

- `name` - назва вина (рядок)
- `price` - ціна (число)
- `rating` - середній рейтинг (число)
